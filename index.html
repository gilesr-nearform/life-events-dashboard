v2 · latest 20 May 2026

Unified application card

A bounding-box card per application that includes its latest messages. Shows three applications by default with a reveal-more affordance.

  • One card per application, with 3 latest messages embedded
  • Card-level “View all N messages” link
  • “Show more applications” reveal pattern
  • Single-column reading layout, max content width 960px
  • Inherits v1’s tokens, bilingual toggle, and responsive rules
